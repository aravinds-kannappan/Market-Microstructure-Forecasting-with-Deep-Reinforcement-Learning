# Market Microstructure Forecasting with Deep Reinforcement Learning

## Quickstart
```bash
python -m venv .venv && source .venv/bin/activate
pip install -r requirements.txt
# (optional) pip install lightgbm stable-baselines3
jupyter lab
```

Put your FI-2010 CSVs in `data/` (or use the provided `LOB_sample.csv`).

## Train a baseline + run demo
Open `notebooks/lob_rl_demo.ipynb` and run cells.
